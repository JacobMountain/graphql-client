# GraphQL Annotations
## @GraphQLClient
#### Example:
```java
@GraphQLClient(
    schema = "./path/to/file.gql",
    mapping = {
        @GraphQLClient.Scalar(from = "ID", to = String.class)
    },
    nullChecking = true,
    implSuffix = "Graph",
    reactive = false,
    dtoPackage = "dto"
)
public interface MyClient {

}
```
